PharmacoDB

I was a developer for Pharmacodb , a web application assembling the largest drug screens into a single database. For more information, visit here.

In particular, I created all the data visualizations using d3.js, and helped design/develop various parts of the UI in Ruby on Rails.

I also helped maintain the database in MySQL, as well as its corresponding API in collaboration with cBioPortal.

PharmacoDB is published in Nucleic Acids Research.

ePlant

For my second year research project, I created a phylogenetic navigator tree for ePlant, a tool to visualize various biological connections given a query gene.

I used d3.js and d3.phylogram.js for the tree, and JavaScript to integrate it into the web app.
